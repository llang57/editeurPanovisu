Description détaillée

Classe de lancement pour jpackage Cette classe wrapper est nécessaire pour lancer une application JavaFX depuis un JAR shaded.

Elle utilise Application.launch() au lieu d'appeler directement main() pour garantir l'initialisation correcte du toolkit JavaFX.
