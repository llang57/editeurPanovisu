◆ affiche()

void editeurpanovisu.VisualiseurImagesPanoramiques.affiche ( )

Affiche la vue panoramique avec les paramètres actuels.

Met à jour la visualisation 3D du panorama en fonction des coordonnées actuelles (longitude, latitude) et du champ de vision (FOV). Cette méthode :

  • Normalise les angles de longitude entre -180° et +180°
  • Limite la latitude pour éviter les distorsions polaires (±80°)
  • Recalcule le champ de vision horizontal en fonction du ratio d'aspect
  • Rafraîchit l'affichage de la scène 3D

Appelée automatiquement lors des changements de position ou de zoom.

◆ imgTransformationImage() [2/2]

static Image editeurpanovisu.VisualiseurImagesPanoramiques.imgTransformationImage ( Image  imgRect,
int  iRapport 
)
static

Transforme et redimensionne une image panoramique rectangulaire.

Convertit une image panoramique équirectangulaire en image réduite adaptée à l'affichage. Utilise le GPU (méthode Bicubic) si disponible, sinon bascule sur une transformation CPU pixel par pixel (projection Mercator).

Rapport de réduction :

  • iRapport = 2 : Divise la taille par 2
  • iRapport = 4 : Divise la taille par 4 (plus rapide)

Exemple d'utilisation :



Image panoramaComplet = new Image("file:panorama.jpg");

// Divise par 4 la taille pour l'affichage miniature

Image miniature = imgTransformationImage(panoramaComplet, 4);

imageView.setImage(miniature);
Paramètres
imgRectImage panoramique rectangulaire source
iRapportFacteur de réduction (2 = moitié, 4 = quart, etc.)
Renvoie
Image transformée et redimensionnée
Voir également
ImageResizeGPU::resizeAuto(Image, int, int, InterpolationMethod)
InterpolationMethod::BICUBIC
