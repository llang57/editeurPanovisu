Description détaillée

NavigateurCarte utilisant JMapViewer (OpenStreetMap pour Java Desktop) Remplace Gluon Maps qui crashait sur Desktop à cause du système de cache.

Solution STABLE basée sur:

  • JMapViewer (bibliothèque OSM éprouvée)
  • SwingNode (intégration Swing dans JavaFX)
  • Pas de WebView, pas de JavaScript
